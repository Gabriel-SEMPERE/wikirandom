# wikirandom
Voici un site assez simple qui utilise l'API de Wikipédia et qui permet d'obtenir des articles aléatoires. 

Vous avez le choix de combien d'article vous voulez, vous aurez pour chaque article ( si c'est possible ) un résumer de l'article. 

Par rapport à https://fr.wikipedia.org/wiki/Sp%C3%A9cial:Page_au_hasard le site permet de voir de quoi parle l'article avant d'aller dessus.

Si aucun article vous plait vous pouvez facilement re générer les articles.
